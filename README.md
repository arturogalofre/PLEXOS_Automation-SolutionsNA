# Energy Exemplar Automation Repository

Welcome to the centralized automation repository for Energy Exemplar. This repository serves as the single source of truth for our internal scripts, tools, and automation workflows.

## Repository Structure

The repository is organized to separate concerns and make it easy to find specific types of automation.

### `cloud/`
Automation, scripts, and resources specific to **PLEXOS Cloud**.
- **`Datahub/`**: Connectors and guides for PLEXOS Datahub.
- **`local orchestration/`**: Scripts for pre/post-processing and local execution workflows.
- **`VM orchestration/`**: Tools and guides for managing VM environments.

### `desktop/`
Automation and tools specific to **PLEXOS Desktop**. (Currently used for desktop-related scripting).

### `docs/`
Documentation, reference materials, and API guides.
- **`PLEXOS_Cloud_CLI_Documentation.md`**: Documentation for the Cloud CLI.
- **`*.pdf`**: Reference guides for API, Data Formats, and Environment setups.
